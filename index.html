Javascript & JQuery Flashcards

Definitions:

JavaScript (JS)

A programming language that performs functions on designated parameters to return a result.

Operators

Language tokens that act similar to functions and have precedence levels and associativity rules.

Variables

A storage location paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.

Variable Naming Conventions

Naming conventions used within JavaScript and jQuery, which includes the use of lowerCamelCase (where a multi-word name begins in lowercase, with the next words joined together without spaces and each of their first letters capitalized).

Functions

A block of code written to obtain one or more data values to be called.

Methods

A function that is accessed as a property of an object.

Arguments

The actual piece of data being used within the function.

Parameters

The variable(s) referring to one of the pieces of data within the function.

Return

The request to perform a specific function.

Chaining Methods

A way of "chaining" methods together in a single statement, or invoking multiple method calls at once.

Strings

A sequence of characters (letters, numbers, or a combination) which form a constant within a function or method.

Booleans

An expression resulting in the value of either "true" or "false".

Undefined

A condition where an expression does not have a correct value.

NaN

A statement meaning "Not a Number", which represents an undefined or unrepresentable value.

Escape

A function that makes a string portable so it can be transmitted across networks.

Alert

A pop-up displaying a designated message.

Comments

Notes about the code that are not interpreted.

jQuery

A library document of JavaScript functions and code that can be referenced in order to use specific pre-built functions.

Attributes

A specification that defines a property of an object or element.
